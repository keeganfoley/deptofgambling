## SCRIPT FOR AUTO-GENERATION

Use `scripts/generate-captions.js` to auto-generate weekly caption options:

```bash

```

Example:
```bash
node scripts/generate-captions.js 6 2025-12-08 2025-12-14
```

This outputs 5 caption variations to choose from.
