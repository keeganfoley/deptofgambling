# Weekly Recap Video - Master Templates

## 4 Slides for Weekly Video

| Slide | File | Description |
|-------|------|-------------|
| **1** | `slide1-weekly-briefing` | Hero overview with P/L, record, win rate, ROI, official DOG statement |
| **2** | `slide2-detailed-breakdown` | Operational metrics, daily performance, bet analysis, by bet type, by sport table |
| **3** | `slide3-fund-performance` | Fund breakdown showing Sharp, Vector, Contra, Catalyst weekly performance |
| **4** | `slide4-season-overview` | Season-to-date summary with cumulative stats |

## Dimensions
- All slides: **1080x1350px** (Instagram portrait)

## How to Create Weekly Recap

1. Copy this folder to `social-images/weekly-recaps/week{N}-{dates}/`
2. Update the HTML files with the week's data
3. Screenshot each HTML at 1080x1350 to generate PNGs
4. Use PNGs for video/carousel

## Data Needed for Each Slide

### Slide 1 - Weekly Briefing
- Week number, date range
- Net P/L, units won
- Record (W-L-P), win rate, ROI
- Official statement highlighting key wins

### Slide 2 - Detailed Breakdown
- Positions, capital deployed, profit factor
- Green days record, avg bet size, units won
- Best day (date + P/L), worst day (date + P/L)
- Avg win, avg loss, biggest win, biggest loss
- By bet type: spreads, moneyline, totals, props (record + P/L each)
- By sport: NHL, NBA, NCAAF, NFL, NCAAB (record, win%, units, P/L, ROI)

### Slide 3 - Fund Performance
- Each fund: Sharp, Vector, Contra, Catalyst
- Weekly P/L, record, units, win rate, ROI per fund

### Slide 4 - Season Overview
- Season-to-date record, P/L, win rate, ROI
- Week-over-week comparison
- Cumulative performance metrics
