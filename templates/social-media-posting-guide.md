## INSTAGRAM FORMAT

### DAILY PICKS - 2 Slides

**SLIDE 1: Picks Summary**
- Title: "NOV [XX] - TODAY'S PICKS" (yellow/gold text on blue gradient background)
- List each pick: "[PICK DESCRIPTION] ([ODDS]) [X]U"
- Total at bottom: "[X] UNITS"
- Logo in corner
- Disclaimer at bottom

**SLIDE 2: Detailed Breakdown**
- Each pick underlined as header
- Bullet points with key reasons (condensed)
- Win probability and Expected ROI for each
- Logo in corner
- Disclaimer at bottom

## CHART IMAGES

### DAILY CANDLESTICK CHART
- Location: `story-images/day-XX-YYYY-MM-DD.png`
- Include with daily report
- Shows: Starting balance, Current balance, ROI, Win Rate
- Visual: Candlestick chart showing daily performance

### WEEKLY CHART (Sundays)
- Location: `weekly-images/week-X-mon-DD.png`
- Posted on Sundays for weekly recap
